Guida utilizzo file E1

Come importare il file .E1 in Ca.R.Pe., struttura interna del file, decodifica, ispezione, e aspetti legali. Tutto quello che succede dopo aver scaricato il tuo estratto contributivo dall'INPS.

Hai bisogno di scaricarlo prima? Vai alla Guida: scaricare il file E1 per il download dal Fascicolo Previdenziale INPS.

Importare il file E1 in Ca.R.Pe.

  1. Apri Ca.R.Pe. PC sul tuo computer.
  2. Dal menu File → Importa… (o Apri… a seconda della versione) scegli il file .E1 appena scaricato.
  3. Il software importa automaticamente tutti i periodi contributivi, l'anagrafica e le retribuzioni note. Vedrai apparire i pannelli Anagrafica, Anzianità, Retribuzioni già popolati.
  4. A questo punto puoi calcolare la pensione (Menu Calcola), simulare scenari futuri (vedi Guida: simulazione pensione) o salvare il risultato come file .EC.

Per il dettaglio operativo dell'import e dei pannelli vedi anche il manuale «Informazioni uso file E1 su Carpe» (PDF).

Struttura del file .E1 (input)

Il file .E1 è un contenitore binario codificato: non è testo leggibile e viene interpretato dal software Ca.R.Pe. PC durante l'import. Le caratteristiche osservabili:

CaratteristicaValore
Estensione.e1 (minuscola)
TipoBinario codificato (non testo)
Naming convention tipicoEC-CODICEFISCALE16.e1
Dimensione tipica50–500 KB
Leggibilità direttaNessuna — non apribile in un editor di testo
CompatibilitàSoftware Ca.R.Pe. PC INPS
Perché è codificato. Il formato non in chiaro serve a ridurre il rischio di alterazione dei dati prima dell'import e a mantenere la lettura all'interno del canale ufficiale. Il file è pensato per essere aperto da Ca.R.Pe. PC, che ne verifica la coerenza durante l'import.

Consultare il contenuto del proprio file E1/EC

Questo manuale include uno strumento che gira interamente nel browser (nessun dato lascia il tuo dispositivo) per convertire il file .E1/.EC nel corrispondente XML leggibile e viceversa, così da consultare la propria posizione e impostare simulazioni personali: Codifica / Decodifica / Modifica .E1 .EC .AG .XML.

A cosa serve. Convertito in XML, il file mostra le stesse informazioni che Ca.R.Pe. presenta a video: anagrafica, periodi contributivi, retribuzioni, datori di lavoro. È utile per rileggere con calma la propria storia contributiva e per preparare ipotesi «what-if» da ricaricare in Ca.R.Pe. sul proprio PC.

Come leggere i dati del proprio estratto

Il modo più semplice e diretto è farsi dare da Ca.R.Pe. stesso una copia leggibile, senza strumenti aggiuntivi.

Salva il file .EC da Ca.R.Pe. È leggibile ed è sufficiente per la quasi totalità dei casi.
Procedura: dopo aver caricato l'E1 in Ca.R.Pe., usa «Salva EC» (oppure menu File → Salva con nome… → tipo: Estratto Conto .EC). Apri il file con un editor di testo: vedrai la tua storia contributiva in chiaro.

In alternativa puoi convertire il file con lo strumento in-browser di questo manuale, che produce lo stesso contenuto in formato XML.

Ambito d'uso

Lo strumento e le informazioni di questa pagina sono pensati per la consultazione e la simulazione sui propri dati. In sintesi:

Usi appropriati (uso personale):

  • Convertire il proprio file E1/EC in XML per leggerlo e archiviarlo.
  • Impostare simulazioni «what-if» sulla propria posizione (es. proiezioni con periodi futuri ipotetici) da ricaricare in Ca.R.Pe. sul proprio PC.
  • Salvare la copia .EC in chiaro che Ca.R.Pe. stesso offre via «Salva con nome → Estratto Conto .EC».
  • Rivolgersi all'INPS per la rettifica di eventuali dati errati: è il canale ufficiale per qualsiasi variazione formale della posizione.

Da evitare:

  • Utilizzare un file modificato come se fosse il documento originale rilasciato dall'INPS, o presentarlo come autentico a enti, patronati, datori di lavoro o terzi. Le modifiche hanno valore solo come simulazione personale.
  • Operare su file E1/EC di altre persone senza il loro consenso: l'estratto contributivo è un dato personale ai sensi del GDPR.
  • Fare un uso commerciale dei contenuti o del software INPS (vedi informativa di distribuzione INPS).

Per qualsiasi variazione con effetto ufficiale sulla propria posizione contributiva fa fede esclusivamente l'INPS.
