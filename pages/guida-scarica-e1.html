Come scaricare il file E1 dall'INPS

Procedura ufficiale per scaricare il file .E1 dall'INPS — l'estratto contributivo personale da importare in Ca.R.Pe. PC per simulazioni e calcoli pensione.

Cos'è il file .E1. È l'estratto contributivo personale in formato dati-INPS, scaricabile dal proprio Fascicolo Previdenziale. Il file è destinato a essere importato in Ca.R.Pe. PC: contiene tutti i periodi contributivi, i datori di lavoro, le retribuzioni dichiarate e i dati anagrafici dell'assicurato. Permette al software di calcolare la pensione e fare proiezioni senza dover digitare manualmente ogni periodo della carriera lavorativa. Tipicamente occupa poche centinaia di KB.

Come scaricare il file E1

La procedura ufficiale richiede l'accesso al portale INPS con SPID/CIE/CNS al proprio Fascicolo Previdenziale del Cittadino.

  1. Accedi al Fascicolo Previdenziale del Cittadino dal portale INPS:
    Fascicolo Previdenziale del Cittadino — INPS Richiede autenticazione SPID / CIE / CNS
  2. Nel menu di navigazione a sinistra, espandi la voce «Posizione Assicurativa».
  3. Nel sotto-menu che compare clicca su «Estratto Conto».
  4. Nella pagina che si apre, in alto, fai click sul link «Estratto Ca.R.Pe.».
  5. Il browser scaricherà un file con estensione .E1 (es. EC-XXXXXXXXXXXXXXXX.e1, dove XXXXXXXXXXXXXXXX è il tuo codice fiscale). Il file occupa tipicamente tra 50 e 500 KB a seconda della lunghezza della tua storia contributiva.
  6. Salvalo in una cartella sicura del tuo computer. Contiene i tuoi dati personali e contributivi: trattalo come un documento riservato.
Privacy. Il file .E1 e l'eventuale .EC di output contengono codice fiscale, data e luogo di nascita, indirizzo di residenza, sesso, e l'intera storia contributiva con datori di lavoro e retribuzioni. Non condividerli via email/cloud non protetti, non pubblicarli online, e cancellali dopo l'uso se lavori su un PC condiviso.
E ora? Per imparare a importare il file .E1 dentro Ca.R.Pe., capire la struttura interna del file e gli aspetti tecnici/legali, vedi la Guida utilizzo file E1.

Cosa fare se il download non funziona

  • «Estratto Ca.R.Pe.» non visibile: alcuni profili (es. iscritti solo a casse professionali) non vedono il link. Verifica con il patronato o l'INPS sportello.
  • Il file scaricato è di 0 KB o pochi byte: la sessione SPID potrebbe essere scaduta. Esci e riautentica.
  • Carpe non riesce ad importare l'E1: scarica l'aggiornamento più recente del software (vedi Download Carpe). Le vecchie versioni non leggono i nuovi formati E1.
  • Errori di import per dati incompleti: spesso la storia contributiva ha lacune (datori di lavoro non più esistenti, periodi pre-1974 mancanti). Vedi Manuale Carpe — Errori per la gestione delle anomalie.
